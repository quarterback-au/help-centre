# Team Members

Invite team members to collaborate in Quarterback.

## User Roles

### Admin
Full access to all features:
- View all data
- Modify settings
- Manage users
- Access billing

### Member
Standard access:
- View all data
- Create and edit activities
- Set up personal alerts
- Generate reports

## Inviting Team Members

1. Go to **Account Settings**
2. Select **Team**
3. Click **Invite Member**
4. Enter their email address
5. Select their role
6. Click **Send Invite**

The invited user will receive an email to set up their account.

## Managing Members

### View All Members
The team page shows:
- Name and email
- Role
- Status (active, invited, disabled)
- Last active date

### Change Role
1. Click on a team member
2. Select new role
3. Confirm change
