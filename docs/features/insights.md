# Insights

AI-powered insights appear throughout the platform, providing context and analysis for activities, announcements, and overall trends.

## Where Insights Appear

### Dashboard Sidebar

AI-generated summary of key trends from the selected period. Highlights patterns in activity volume, sentiment shifts, and notable events.

### Announcement Panel

Market reaction analysis including:

- Trading context (volume, price movement)
- Sentiment summary from linked activities
- Key themes identified in discussion

### Activity Detail

Contextual insights showing related announcements, author patterns, and market timing.

### Reports

AI-generated overview section summarizing the reporting period.

## How Insights Work

Insights are generated using large language models trained to understand investor relations context. The system analyzes:

- Activity content and metadata
- Market data (price, volume, timing)
- Historical patterns
- Sentiment distribution

Insights reference commodity movements and benchmark performance when relevant to your company's market settings.
