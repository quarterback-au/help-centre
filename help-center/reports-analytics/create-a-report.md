# How to Create a Report

Generate comprehensive investor relations reports that combine sentiment analysis, market data, and activity summaries.

## Steps

1. Navigate to **Reports** from the main sidebar
2. Click **Create Report**
3. Choose a report template or start from scratch:
   - **Weekly Summary** — Activity and sentiment for the past 7 days
   - **Monthly Board Report** — Comprehensive monthly overview
   - **Custom** — Select your own date range and sections
4. Set the **date range** for the report
5. Select which **sections** to include:
   - Executive Summary
   - Sentiment Analysis
   - Market Performance
   - Announcement Impact
   - Media Coverage
   - Social Media Activity
6. Click **Generate Report**
7. Review the report and make any edits
8. **Export** as PDF or share via a link

## Tips

- Reports can be scheduled to generate automatically (weekly or monthly)
- Saved reports can be duplicated and customised for different audiences
- Use the **Compare** feature to benchmark against previous periods

:::tip
For board presentations, use the Monthly Board Report template — it includes executive-friendly charts and summaries.
:::
